WELCOME TO THE KNIGHT'S TOUR!!

THE RULES OF THE GAME ARE AS FOLLOWS:

  • The Knight can only move in an L shape,ie,two squares in one direction (horizontally or vertically) and then one square perpendicular to that direction.
  • Each square can be visited only once
  • The game is won when all 64 squares have been visited.

    • HOW TO PLAY

  • Select a square where you want the tour to start
  • Make legal moves until you either reach a dead end(lose) or you have visited all the squares on the board(win)
  • Use the hint button to find the next legal move
  • You can reset the tour at any point
